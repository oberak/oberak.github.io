Create a Blog site using Hexo

0. Prerequisite

  • Install Node.js
  • Install Yarn (Optional)
  • Install VSCode or Other Markdown Editor
  • Knowledge of Markdown

1. Install hexo-cli

2. Create blog-project

  • Refer to Hexo, Hexo Commends

Project structure

3. Start server

  • localhost:4000

4. Add post

  • Post file created: /source/_posts/Hello-Hexo.md
  • Modify the post file: Hexo Markdown Syntax

5. Build

  • Generate html files under /public/ folder

6. Deploy to GitHub Pages

Create GitHub repository

  • Repository name: username.github.io (‘username’ is GitHub user name. It’s your blog domain)
  • Install ‘hexo-deployer-git’

Modify config

  • Refer to Hexo Configuration
  • Use your github.com username instead of ‘username’

  • Delete ‘Hello World’ post
    delete the post file: /source/_posts/hello-world.md
    and clean

  • deploy to git

Open username.github.io site on your browser(Use your github.com username.)
